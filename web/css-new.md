# white-space:pre;

> 出处：腾讯邮件。
> 作用：显示原空格

```css
    <span class="Apple-tab-span" style="white-space:pre">        </span>
```

# width: -webkit-fit-content;

> 出处：
> 作用：自动修改文字大小，填充整个空间

```css
    .line {
        width: -webkit-fit-content;
        width: fit-content;
        margin: auto;
        }
```
