## terminal cowsay

```
brew install cowsay

brew install fortune

gem install lolcat

gem install artii

# start

cowsay -f vader "hello"

fortune | cowsay -f vader "hello"

artii “Hello world!”

fortune | cowsay -f vader "hello" | lolcat

artii “Hello world!”
```
